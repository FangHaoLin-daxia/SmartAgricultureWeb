# Smart Agriculture Management System — Front-End Web Template

One-stop digital solution for agricultural production, monitoring, and management

## Introduction

`Smart Agriculture Management System` is a mid-backend front-end solution, implemented using `HTML`, `CSS`, and `JavaScript`, and is currently still under development.

## Device compatibility status

Already compatible with `Windows`, `MacOS`, and `Linux`

Not yet adapted for small screens like `Android` and `iOS`
