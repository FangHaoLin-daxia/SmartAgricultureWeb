# 智慧农业管理系统——前端网页模板

一站式农业生产、监测、管理数字化解决方案

## 简介

`智慧农业管理系统`是一个中后台前端解决方案，基于 `HTML`、`CSS`、`JavaScript` 实现,目前仍在开发。

## 设备适配情况

已经适配 `Windows`、`MacOS`、`Linux`

暂未适配 `Android`、`iOS` 等小屏设备
